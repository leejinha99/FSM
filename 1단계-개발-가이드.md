## Apps Script 배포

1. [script.google.com](https://script.google.com) → 새 프로젝트
2. `apps-script/Code.gs` 내용 붙여넣기
3. **프로젝트 설정 → 스크립트 속성 추가**
   - 속성명: `SHEET_ID` / 값: 스프레드시트 URL의 `/d/` 뒤 ID
4. **배포 → 새 배포**
   - 유형: 웹앱
   - 실행 계정: **나(Me)**
   - 액세스 권한: **모든 사용자 (Anonymous)**
5. 배포 URL 복사 → `.env`의 `VITE_SCRIPT_URL`에 붙여넣기

## 1단계 주요 동작 흐름

```
로그인
  → 기사: /calendar (내 캘린더)
  → 관리자: /admin (3단계 예정)

캘린더
  ├─ 월간 뷰: 날짜 탭 → 당일 방문 목록 표시
  ├─ 주간 뷰: 가로 스크롤, 요일별 카드
  └─ 목록 뷰: 오늘 이후 일정만 날짜 그룹

방문 등록 (+ 버튼)
  1. 기본 정보 (날짜/시간/알림/학교/방문유형)
  2. 대상 장비 선택
     - 필터교체/점검: 전체 자동 선택, 개별 해제 가능
     - AS: 특정 장비 선택
     - 설치: 신규 장비 정보 입력
  3. 작업 내용
  4. 사용 부품 (재고 부족 경고 포함)
  5. 청구 금액 (비계약 학교만 표시)
  6. 결제 방식 (카드/세금계산서)
  저장 → Apps Script → 방문기록 + 부품사용 시트 동시 기록
```

---

## 2단계 예정 항목

- `[기사] 내 차량 재고` — 창고별 현재고, 안전재고 경고
- `[기사] AS 접수함` — 배정된 AS 목록, 일정 잡기
- `[공통] 학교 목록·상세` — 검색/필터, 장비별 이력

## 3단계 예정 항목

- `[관리자] 전체 캘린더` — 기사별 색상, 필터
- `[관리자] AS 접수 등록` — 학교+기사 배정
- `[관리자] 전체 목록` — 다중 필터, 엑셀 내보내기
- `[관리자] 재고 총괄` — 4창고 현황, 입고/이동/불량교체
